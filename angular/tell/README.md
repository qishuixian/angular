## 通讯录项目

本项目使用angular-cli模仿书籍《揭秘Angular》通用例子


- 运行 `npm install` 或者 `yarn install` 安装依赖
- 运行 `npm start` 构建并启动本地Server
- 浏览器打开 http://localhost:4200 即可访问

> 服务端渲染版本请运行 `git checkout ssr` 切换至 `ssr` 分支
